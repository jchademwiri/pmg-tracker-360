# Audit Reports

This directory contains focused, standalone audit reports for Tender Track 360.

Each file is self-contained with its own context, issues, and actionable suggestions. Pick any file to start fixing that area.

## Structure

| File | Area |
|------|------|
| `00-overview.md` | Executive summary + priority matrix + index |
| `01-admin-app.md` | Admin Console (all modules) |
| `02-tracker-layout-auth.md` | Tracker layout, auth, onboarding |
| `03-tracker-landing.md` | Landing/marketing pages |
| `04-tracker-dashboard.md` | Tracker dashboard |
| `05-tracker-tenders.md` | Tenders module (CRITICAL) |
| `06-tracker-clients.md` | Clients module |
| `07-tracker-projects-po.md` | Projects + Purchase Orders |
| `08-tracker-calendar-reports.md` | Calendar + Reports (CRITICAL) |
| `09-tracker-settings-billing-org.md` | Settings, Billing, Organization |
| `10-database-schema.md` | Database schema + improvements |
| `11-cross-cutting.md` | Error handling, loading, toasts, testing |

## How to Use

1. Start with `00-overview.md` for the full picture
2. Pick a file based on what you want to fix
3. Each file lists the specific code that needs modification
4. Focus on 🔴 Critical items first
