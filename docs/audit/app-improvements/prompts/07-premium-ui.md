# 07 – Premium SaaS UI Direction

**Purpose:** Recommend a premium design direction for the Tracker app. The design must feel suitable for a serious business operations platform, not a basic admin panel.

**Depends on:** [01-codebase-audit.md](./01-codebase-audit.md)
**Feeds into:** [06-mobile-ux.md](./06-mobile-ux.md), [10-deliverables-roadmap.md](./10-deliverables-roadmap.md)

---

## Task

Define the visual design direction and component design system for the Tracker app.

---

## A. Layout and Typography

- Clean dashboard layout patterns
- Typography hierarchy (headings, body, captions, labels)
- Spacing system (margins, padding, grid)
- Section hierarchy and visual grouping

## B. Cards and Containers

- Modern card designs for KPIs, summaries, and detail sections
- Card variants (stat cards, progress cards, alert cards, action cards)
- Container styles (panels, sections, wells)
- Border, shadow, and elevation usage

## C. Colour System

- Primary and secondary colour palette
- Status colours (success, warning, error, info)
- Tender status colour mapping
- Project status colour mapping
- PO status colour mapping
- Background and surface colours
- Text colour hierarchy

## D. Buttons and Actions

- Button hierarchy (primary, secondary, tertiary, ghost)
- Button sizes and states (default, hover, active, disabled, loading)
- Icon buttons and icon-only buttons
- Floating action buttons for mobile

## E. Status Indicators

- Status badge design and colour mapping
- Progress indicators (bars, rings, percentages)
- Risk indicators
- Priority labels
- Timeline/activity log components

## F. Icons and Imagery

- Consistent icon system (which icon library)
- Icon sizing and alignment
- Empty state illustrations
- Loading state animations

## G. Components to Design

- KPI cards
- Status badges
- Tender pipeline board
- Project progress card
- PO status card
- Deadline alert card
- Register tables
- Mobile register cards
- Activity timeline
- Follow-up log
- Extension history
- Document checklist
- Empty states
- Quick action panel
- Filter drawer
- Command palette / command menu

## H. Overall Feel

The design should feel:
- Premium and professional
- Clean and uncluttered
- Information-dense but readable
- Consistent across all views
- Modern (2024+ SaaS design trends)
- Suitable for daily operational use
