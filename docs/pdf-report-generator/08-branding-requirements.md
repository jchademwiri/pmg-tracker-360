# PDF Report Branding Requirements

Status: Draft
Owner: PMG Tracker 360
Created: 2026-06-25

## Purpose

Every generated PDF report must carry two levels of branding.

1. The customer's organization branding must be the primary branding.
2. Tender Track 360 or PMG Tracker 360 must appear as small footer branding.

## Organization Branding

Every Tracker report must show the organization name in the report header. Where available, the header should also show the organization logo, email, phone, and address.

The organization branding must be more prominent than the product branding.

Minimum header requirement:

- Organization name
- Report title
- Generated date

If the organization logo is missing, the report must still generate and display the organization name as text.

## Product Footer Branding

Every PDF report must include small product branding in the footer on every page.

Recommended footer text:

- Generated by Tender Track 360
- Page X of Y

The product footer branding must be subtle, small, and neutral. It should not compete with the customer's organization branding.

## Tracker Reports

Organization branding is mandatory for all Tracker reports, including tender reports, project reports, purchase order reports, delivery reports, client reports, and document reports.

## Admin Reports

Admin reports may use platform branding. If an admin report is scoped to a specific organization, the report should also include that organization's name or branding.

## Data Model Requirement

The shared report data model should include a branding object with these fields:

- organizationName
- organizationLogoUrl
- organizationEmail
- organizationPhone
- organizationAddress
- productName
- productFooterText

The app data builder should resolve branding and pass it to the PDF renderer. The PDF renderer should not perform database lookups.

## Acceptance Criteria

- Organization name appears in the report header.
- Organization logo appears when available.
- Missing logo does not break PDF generation.
- Report title appears in the header.
- Generated date appears in the header.
- Tender Track 360 or PMG Tracker 360 appears in the footer.
- Footer branding appears on every page.
- Page number appears on every page.
