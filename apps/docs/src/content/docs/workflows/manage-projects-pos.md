title: Manage Projects and POs
description: Track delivery execution and purchase orders after tender activity.
---

## When to use this workflow

Use this flow once tender work transitions into execution and commercial tracking.

## Workflow

1. Open or create a project linked to the relevant tender.
2. Validate project status and ownership.
3. Capture and track purchase orders against the correct project.
4. Keep PO and project statuses aligned with real-world progress.

## Good practices

- Treat project status as the source of truth for delivery state.
- Keep PO references complete and standardized.
- Review exceptions weekly (missing links, stale statuses).
